# sh_LoginTest

 첫 화면에서 ID(e-mail 형식), PW(6자리 이상) 입력 후 Create하면 계정이 생성됩니다.
 
 Ex) 1234@abc.def / 123456
 
 생성한 계정으로 로그인하면 채팅 화면으로 이동합니다.
